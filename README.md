Zadatak 2:
Koristeći Rx.NET i YouTube API, prikupiti komentare za zadate video snimke (prosleđivanjem
identifikatora video snimaka). Rx vrši osnovno mapiranje komentara i emituje ih kao poruke
aktorima. Aktori čuvaju komentare po video snimku kao interno stanje i implementiraju Sentiment
analizu koristeći SentimentAnalysis.NET ili ML.NET biblioteke, računajući sentiment na nivou
videa. Web server prima zahteve sa identifikatorima video snimaka i prevodi ih u poruke aktorima.
Prikazati dobijene rezultate.
Dokumentacija dostupna na linku: https://developers.google.com/youtube/v3/docs
